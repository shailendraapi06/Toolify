# Toolify

Toolify is a modern, lightweight, and privacy-focused collection of online utility tools built using **HTML, CSS, and Vanilla JavaScript**.  
All tools run completely in the browser with no backend or server dependency.

---

## Live Demo

Live website:  
https://mytoolify.netlify.app  


---

## About the Project

Toolify provides a wide range of everyday tools including calculators, text utilities, image tools, PDF tools, and fun tools.  
The main goal of this project is to deliver fast, secure, and easy-to-use tools without relying on frameworks or external services.

---

## Key Features

- Fully client-side (no backend, no database)
- Privacy-first approach (no user data stored)
- Responsive design for mobile and desktop
- SEO-friendly HTML pages
- Clean and scalable structure
- Built without any frameworks (no React, no Bootstrap)

---

## Tool Categories

- Calculators  
- Text Tools  
- Image Tools  
- PDF Tools  
- Utility Tools  
- Fun Tools  

Each tool is designed to work independently and load instantly in the browser.

---

## Technology Stack

- HTML5 for structure and SEO
- CSS3 for layout and responsiveness
- Vanilla JavaScript for interactivity
- Netlify for hosting and deployment

---

## Running the Project Locally

1. Download or clone the repository
2. Open `index.html` in any modern browser
3. No installation or setup required

---

## Deployment

The project is deployed using Netlify.

Steps:
- Upload the project folder to Netlify  
  or  
- Connect the GitHub repository and deploy automatically

Make sure `index.html` is present in the root directory.
